We can use counters to keep track of information in web apps. A counter is just a variable that holds a changing number. We usually start counters off at 0:

counter = 0

We called the variable counter, but we could have called it anything. We can add 1 to the counter by using this syntax:

counter = counter + 1

It might seem weird, but the above line is just saying "Take what's in counter, add 1 to it, then save the result back into counter.

We can use the same basic syntax inside of a rails app. Suppose we were using a cookie to keep track of numbers coming from the params hash:

cookies["counter"] = cookies["counter"].to_f + params["number"].to_f

We just substitued the cookie for the variable and converted the params hash value into a number. This won't work if cookies["counter"] doesn't exist yet, though. We need to make sure the we just set it to params["number"].to_f then:

if cookies["counter"]
  cookies["counter"] = cookies["counter"].to_f + params["number"].to_f
else
  cookies["counter"] = params["number"].to_f
end
