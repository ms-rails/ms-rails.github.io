Building up to starter generators

Using starter generators

To use the starter generators command, we'll first need to add the gem to our gemfile. Open up the file called `Gemfile` and paste in this line anywhere on the page.

gem 'starter_generators', :git => 'git://github.com/arjunvenkat/starter_generators'

Then, in your app folder in terminal, type in the command: 

bundle install

This loads the gem into your app.

We can use starter generators with this general syntax:

rails generate starter:basic [model_name] [column_name]:[column_data_type]

It works just like the model generator we learned about earlier. Here's a concrete example:

rails generate starter:basic Fruit name:string rating:integer

This one line creates all the routes, controllers and views you need to CRUD out a whole resource!
