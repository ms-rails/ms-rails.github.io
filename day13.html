Working with arrays

An array is a list that looks like this:

stuff = [3, "hello", 4.8, false, 89]
end

We can store lot's of different things inside an array.

We pull out items by their poistion in the array. We have to subtract by one, though. For example, if we want to pull out "hello" from the above array, we would use:

stuff[1]

We can loop through arrays with the each method:

colors = ['red', 'yellow', 'blue']

colors.each do |color|
  puts "#{color} is a primary color"
end

The code between the do and the end gets run for each item in the list. The variable between the two | | is a placeholder for each list item. So, the first time the loop is run, color stands for 'red', the second time color stands for 'yellow' and the third time, color stands for 'blue.

Creating an index view

The index view uses the each loop, but we need to make sure we wrap each line <% %> tags. Notice that there's no equal sign on the first or last line of the loop. That's because those lines are just Ruby code that we want to run - we don't want those lines to show up in the HTML.
