For next week we'll start working with cookies. Watch and type along with the video below. I've included some notes to help you along the way.

We can use cookies to temporarily store information in web apps. Cookies work kind of like the params hash. Suppose we wanted a cookie to hold a user's name. We could set it in the controller with:

cookies['name'] = "Arjun"

Now, the cookie should be stored in the browser. We can access the cookie with:

cookies['name']

just like we would access something in the params hash. The key inside the [ ] can be any string we want. For example, if we wanted to set a user's birthday, we could use:

cookies['birthday'] = "1/25/86"

If we wanted to set a cookie with a value from the params hash, we could use:

cookies['name'] = params['name']

Sign-in is just the beginning of what we'll be able to eventually do with cookies.
